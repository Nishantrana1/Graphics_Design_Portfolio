# Graphics Design Portfolio

A professional and immersive portfolio website showcasing graphic design and video editing work. 
Designed with a focus on modern web aesthetics, dynamic interactions, and a cohesive artistic identity.

## Features
- **Dynamic Interactions**: Smooth animations and hover effects that bring the portfolio to life.
- **Modern Aesthetics**: Curated color palettes, sleek typography, and glassmorphism elements.
- **Responsive Design**: Fully responsive layout that looks great on all devices.

## Technologies Used
- HTML5
- CSS3 (Vanilla)
- JavaScript

## Setup
To view the portfolio locally:
1. Clone this repository:
```bash
git clone https://github.com/Nishantrana1/Graphics_Design_Portfolio.git
```
2. Open `index.html` in your web browser.
